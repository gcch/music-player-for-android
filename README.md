## Overview

Android 向けの広告のない、シンプルな音楽プレイヤーが欲しいと思って作り始めた音楽プレイヤーアプリ。

基本的な音楽再生、タグ情報表示くらいまではできたはずですが、途中で挫折したので、どこまでできているかは失念。


## Development Environment

Android Studio にて開発。
